    XDC (约束文件)   UG903
        都处于Language Templates 下的XDC文件夹中

    1. 时钟
       
        clock constraint 
       
        create_clock -name <clock_name> -period <period> [get_ports <port_name>]
            
 
    2. 引脚位置
    
        IO assignment , 
        
                set_property PACKAGE_PIN <pin name> [get_ports <port>] 
                
    3. 电平标准  ( UG899 )
    
        IO standard 
       
                set_property IOSTANDARD <IO standard> [get_ports <ports>]
       
    
    4. 比特流优化配置   ( UG908 )
        
       比较常用 : 
       
         1. 配置速度 
         
                 使用一个外部晶振来作为配置的时钟,可以加快速度,适用于主机模式 ( UG908 )
                        
                    set_property BITSTREAM.CONFIG.CONFIGRATE 50 [current_design]
         
         2. 配置第三方flash芯片时,选择spi总线 x2 或 x4  ( UG908 )
                 
                set_property BITSTREAM.CONFIG.SPI_BUSWIDTH 4 [current_design]
         
         3. 这个配置,取决于flash芯片怎么连线, 不同模式有不同的读写,看引脚怎么连. 开发板上用spix4  (UG470)    
           
                set_property CONFIG_MODE SPIx4 [current_design]
